# Ivey Publishing — Case Submission

This folder contains the case study and teaching note for submission to Ivey Publishing.

## Documents

1. **`case-study.md`** — The case document (target: ≤8 pages body, ≤15 pages with exhibits)
2. **`teaching-note.md`** — The accompanying teaching note

## Submission Format

Per Ivey guidelines, final submission requires:
- Case and teaching note as separate Microsoft Word documents (.doc or .docx)
- 11-point Times New Roman font
- Exhibits at end of each document
- CMOS 17th edition footnote citations

## Case Type

**General Experience** — based on the author's first-hand experience building and operating VZLA Sports Elite.
